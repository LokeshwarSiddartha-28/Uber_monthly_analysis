# 🚖 Uber_monthly_analysis | Power BI

## 📌 Project Overview
This project analyzes Uber trip bookings for **June 2024** using Power BI. The dashboard provides insights into booking trends, trip patterns, revenue metrics, and customer behavior through interactive visualizations and KPI cards.

---

## 📊 Dashboard Features

- Total Bookings
- Total Booking Value
- Average Trip Distance
- Average Trip Time
- Booking Trends
- Peak Booking Hours
- Vehicle Type Analysis
- Payment Method Analysis
- Trip Distance Analysis
- Dynamic Date Filtering
- Interactive Navigation
- Average Diesel Spendings

---

## 🛠 Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX
- Data Modeling
- Excel

---

## 📈 DAX Measures Created

- Total Bookings
- Total Booking Value
- Average Booking Value
- Average Trip Distance
- Average Trip Time
- Total Trips
- Dynamic KPIs

---

## 📊 Key Insights

- Analysis covers **June 2024** trip data.
- Peak booking hours were **1:00 PM – 6:30 PM**.
- **Monday** and **Saturday** recorded the highest bookings.
- **Thursday** had the lowest booking volume.
- Dashboard supports interactive filtering and drill-down analysis.
